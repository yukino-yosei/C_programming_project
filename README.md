# 학급 성적 관리 프로그램
6개의 반의 각 학생 수를 입력받은 뒤 성적을 입력받아 석차, 등급, 학급 성적 지표 등을 계산해주는 프로그램입니다.
- - -
## 요구사항
C언어 컴파일러 및 구동기가 필요합니다. (cpp 포함)
- - -
## 기능
1. 각 첫줄에 각 반의 N명의 학생 수를 알려준 후
2. N줄에 걸쳐 학생들의 성적을 입력합니다. (국어 - 영어 - 수학 - 한국사 - 과학탐구 - 사회탐구 순서)
3. 1 ~ 2 과정이 6번 반복됩니다.

4. 다음 줄부터는 명령어를 입력받습니다
  - 1. 학생 성적 업데이트
  - 2. 학생 과목 석차 계산
  - 3. 학생 과목 등급 계산
  - 4. 학급 전체 성적 지표 계산
  - 5. 프로그램 종료
- - -
명령어에 5가 입력될 시 프로그램이 종료됩니다.
